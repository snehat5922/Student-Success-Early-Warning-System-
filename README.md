# 🎓 Student Success & Early Warning System

An AI-powered analytics platform that identifies students at risk of academic failure using attendance, LMS activity, assignment completion, and quiz performance.

## Features

- Student Risk Prediction
- Academic Performance Analytics
- Attendance Monitoring
- LMS Engagement Analysis
- SHAP Explainability
- Intervention Recommendations
- Streamlit Dashboard
